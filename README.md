## 페이지별 상세페이지별 상세

1. 회원가입 페이지

   - [x] 이메일 형식 체크
   - [x] 비밀번호 체크
   - [x] 공란시 가입불가
   - [x] 이메일 중복 체크
   - [x] 닉네임 중복 체크
   - [x] 중복 체크 완료후 가입 버튼 생성

2. 로그인 페이지

   - [x] 가입된 아이디/비번 일치시 로그인

3. 메인 페이지(게시글 목록 페이지)

   - [x] 게시글 목록 노출
   - [x] 게시글 하나는 작성자 이메일, 닉네임, 유저 프로필 이미지, 작성 날자  텍스트 내용으로 구성
   - [x] 게시글 하나를 클릭 시, 게시글 상세 페이지 모달창 오픈
   - [x] 스크롤 맨 밑으로 자동 이동
   - [x] 실시간 포스팅 랜더링
   - [x] 입력 완료후 입력창 리셋  
   - [x] 유저 프로필 확인

4. 채팅방
   - [x] 실시간 채팅 랜더링

   - [x] 댓글 작성
